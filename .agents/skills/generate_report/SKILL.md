# 股市投資週報生成指南

此 Skill 指引 AI 代理如何依據使用者的投資策略，自動收集市場新聞並生成結構化的 Markdown 股市週報。

## 投資核心策略
- **價值投資與前瞻性**：不做短線投資，標的持有 3 年以上。
- **台美股為主**：排除所有加密貨幣（不關注加密貨幣，不包含任何加密貨幣消息）。
- **貨幣單位**：台股以台幣 (TWD) 計算，美股以美金 (USD) 計算。

---

## 執行流程

### 步驟 1：讀取與篩選標的
- 讀取專案根目錄下的 [watchlist.json](file:///f:/VibeCodingProject/InvestmentTool/watchlist.json)。
- 篩選出所有 `ticker` 不含 `-USD` 且 `tags` 不包含 `"加密貨幣"` 的標的。
- 區分出哪些是「持有中」（`holding` 為 `true` 或 `shares > 0`）與「未持有/僅關注」的標的。

### 步驟 2：讀取歷史報告作為參考
- 檢查 `reports/` 資料夾，尋找最新的兩份週報檔案（例如 `report_YYYY-MM-DD.md`）。
- 讀取其內容，理解上期與上上期的配置比例、投資觀點與信心指數，以確保新一期報告策略的連貫性。

### 步驟 3：收集重大新聞與情報
- 針對關注的台美股標的，使用網頁搜尋功能（`search_web`）尋找過去一個月內影響其波動的重大新聞。
- **重要來源**：Yahoo股市、TradingView、PTT Stock版、鉅亨網、Finviz、華爾街日報等。
- 對於沒有重大新聞的標的，無須刻意產出內容，以避免報告過於膨脹。
- 將重複或相似的情報進行彙整歸納，避免雜亂無章。
- 對於每一條情報，必須註明**來源出處**。

### 步驟 4：分析看漲與看跌論點
- 針對情報進行分類：
  - **看漲 📈** 或 **看跌 📉** 或是 **持平 ⚖️**。
  - 必須區分 **短期論點**（未來幾周到幾個月的波動）與 **長期論點**（3年以上的價值與基本面變化）。

### 步驟 5：評估其他潛力標的
- 尋找目前資料庫中尚未關注，但具有前瞻性與價值投資潛力的 1-2 個新標的，說明推薦原因。

### 步驟 6：規劃 100% 新資金配置建議
- 假設當前有 100% 新資金，建議如何分配在這些標的上（不限於資料庫內已關注的標的）。
- 以 Markdown 表格呈現，包含：`標的代號`、`建議配置比例 (%)`、`國家/貨幣`、`信心指數 (%)`、`配置原因與風險說明`。

### 步驟 7：給出持股賣出/減碼建議
- 檢查當前持有股票（`holding` 為 `true` 的標的），給出是否建議賣出/減碼以及賣出比例的建議。若無，則說明維持 Buy and Hold 的原因。

### 步驟 8：寫入報告與通知
- 將生成的報告寫入 `reports/report_YYYY-MM-DD.md`。`YYYY-MM-DD` 應使用當前生成報告的 Local 日期。
- 確保所有輸出、註解均為**繁體中文**。
- 完成後通知使用者，並附上新報告的連結。
